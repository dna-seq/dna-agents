# Scout papers for annotation modules

Find and triage genomics papers suitable for creating just-dna-lite annotation modules.

1. Read `.claude/agents/paper-scout.md` and follow it (or Task `paper-scout`).
2. Classify each paper: suitable candidate-gene / GWAS(+supplements) / pharmacogenomics vs unsuitable (reviews, PRS-only, expression-only).
3. For suitable papers: PMID, type, genes, where SNP data lives (main vs supplements), open-access status, recommended next step (`@module-creator` / create-module).
4. Never invent PMIDs. Prefer open-access sources with attachable full text + supplements.
